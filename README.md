# Kingdom Journey Website

This is a browser-native website built with:

- `index.html` for page content
- `styles.css` for colors, layout, and responsive design
- `script.js` for navigation, the mobile menu, and the contact form

## Edit the site

Open this folder in Codex and describe the change you want. For example:

- Change the home page headline
- Add a new ministry update
- Replace the donation link
- Adjust the colors or fonts
- Add An's photo as `an_photo.jpeg`

## Preview the site

Open `index.html` in a web browser. No build step or package installation is required.

## Photo

The page expects a photo named `an_photo.jpeg` in this folder. Until it is added,
the home page displays its existing placeholder.
